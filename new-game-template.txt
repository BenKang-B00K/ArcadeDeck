=============================================
  ArcadeDeck - 새 게임 추가 가이드
=============================================

[Step 1] 썸네일 이미지 준비
--------------------------------------------------
- WebP 형식만 사용
- public/images/ 폴더에 파일 추가
- 파일명 예시: my-game-banner.webp

[Step 2] games.json에 게임 데이터 추가
--------------------------------------------------
- 수정 파일: src/data/games.json
- new-game-template.json 을 복사해서 값만 수정
- id는 기존 마지막 ID + 1 (문자열. 예: "16")
- slug는 URL에 들어갈 영문 소문자-하이픈 (예: "my-game")
- 나머지 필드 설명은 아래 참고

[Step 3] 검증 & 빌드
--------------------------------------------------
npm run validate     ← 데이터 검증만 실행
npm run build        ← 검증 + sitemap 생성 + 빌드 (한번에)
npm run deploy       ← GitHub Pages 배포

※ sitemap.xml은 빌드 시 games.json에서 자동 생성됩니다.
  수동으로 수정할 필요 없음.

=============================================
  필드 설명
=============================================

[필수 필드]
--------------------------------------------------
id              숫자 문자열. 기존 마지막 ID + 1 (예: "16")
slug            URL 경로용 영문 소문자-하이픈 (예: "galaxy-launch")
title           영문 게임 제목
description     영문 한줄 설명 (1~2문���)
fullDescription 영문 상세 설명 (3~6문단 권장)
thumbnail       썸네일 경로 (예: "images/my-game.webp")
genres          장르 배열 (아래 기존 장르 목록 참고)
gameUrl         게임 iframe URL (예: "https://my-game.pages.dev/")
status          "PLAYABLE" 또는 "IN PRODUCTION"
                IN PRODUCTION이면 사이트에 표시되지 않음

[선택 필드]
--------------------------------------------------
language        언어 배지 (KO, EN, KO/EN). 없으면 배지 미표시
isOriginal      직접 만든 게임이면 true (Original 배지 표시)
aspectRatio     화면 비율 (기본값 "16/9". 세로형은 "9/16")

[한국어 번역] (강력 권장)
--------------------------------------------------
titleKo           한국어 제목
descriptionKo     한국어 한줄 설명
fullDescriptionKo 한국어 상세 설명
controlsKo        한국어 조작법
tipsKo            한국어 팁
loreKo            한국어 배경 스토리
featuresKo        한국어 특징 배열

[게임 상세 정보]
--------------------------------------------------
controls   영문 조작법 (예: "Mouse Click: Attack / WASD: Move")
tips       영문 전략 팁 (2~4문장)
lore       영문 배경 스토리 (2~4문단)
features   특징 태그 배열 (5~8개 권장)
           ※ 모바일 지원이면 "Web & Mobile Friendly" 포함
           ※ PC 전용이면 "PC Only" 포함

[리더보드 설정]
--------------------------------------------------
leaderboard.title          리더보드 제목
leaderboard.primaryLabel   주 점수 이름 (예: "Score", "Stage", "Waves")
leaderboard.primaryUnit    주 점수 단위 (예: " pts", "". 없으면 빈 문자열)
leaderboard.secondaryLabel 부 점수 이름 (없으면 빈 문자열)
leaderboard.secondaryUnit  부 점수 단위 (예: "Lv.", "x")
leaderboard.dualSort       부 점수 2차 정렬 여부 (true/false)
leaderboard.subSortAsc     동점 시 부 점수 낮은게 유리? (true/false)

리더보드 예시:
  점수만:        primaryLabel:"Score"  dualSort:false
  점수+레벨:     primaryLabel:"Score"  secondaryLabel:"Level"   dualSort:true  subSortAsc:false
  점수+시도횟수: primaryLabel:"Score"  secondaryLabel:"Merges"  dualSort:true  subSortAsc:true

=============================================
  게임 내 점수 전송 (게임 코드에서 구현)
=============================================

게임 종료 시 아래 코드를 호출하면 ArcadeDeck이 자동으로 점수를 수신합니다:

  window.parent.postMessage({
    type: "GAME_SCORE",
    score: 12500,         // 주 점수 (0 ~ 10,000,000)
    subScore: 15          // 부 점수 (선택)
  }, "*");
